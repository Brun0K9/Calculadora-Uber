
# 🚖 Calculadora para motorista de aplicativos

Aplicação web desenvolvida em React para ajudar motoristas de aplicativo a calcular o lucro diário com base em faturamento, custos e tipo de veículo.

---

## 💡 Funcionalidades

✅ Cálculo automático de lucro diário  
✅ Inserção de:
- Faturamento do dia  
- Gasto com combustível  
- KM rodados  
- Outros custos  

✅ Cálculo de custo do carro:
- 🚗 Financiado (parcela mensal / dias trabalhados no mês)  
- 🚘 Alugado (valor semanal / dias da semana)

✅ Histórico por dia da semana  
✅ Percentual de lucro (%)  
✅ Interface simples e responsiva  

---

## 🧠 Como funciona

A aplicação calcula:

- **Custo total diário**
- **Lucro líquido**
- **Porcentagem de lucro**

### 📌 Fórmulas utilizadas:

**Carro financiado:**

dias no mês = (dias por semana × 30) ÷ 7
custo diário = parcela mensal ÷ dias no mês

**Carro alugado:**

custo diário = aluguel semanal ÷ dias trabalhados

**Lucro:**

lucro = faturamento - custos
